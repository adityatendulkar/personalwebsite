Bulk Singles

A bulk cost searcher and optimiser for MTG singles orders that helps find lower-cost combinations across vendors.

Python Flask Desktop packaging

What it does.

Bulk Singles compares listings from mtgsingles.co.nz, applies vendor and shipping costs, and searches for a cheaper way to fulfil a bulk order.

The project ships as both a local web app and a desktop app, so it can be used without setting up a full development environment each time.

Key capabilities

  • Compares card availability and pricing across multiple stores.
  • Optimises for total order cost instead of only per-card pricing.
  • Supports desktop downloads through GitHub Releases.
